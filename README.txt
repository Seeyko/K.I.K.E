TOUCHE :
- Touche fléchées | pour se deplacer.
- ESPACE | pour changer de salle
- E pour actionner les boutons/levier ou plaques de pression.

Prérequis :

Java a jour version minimum disponible ici : 

http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

Ouvrir un terminal dans le dossier et executer :

java -jar K.I.K.E.jar

Pour un meilleur gameplay veuillez lire "Introduction.txt"
